
# 皇室战争小程序
### 项目说明：
微信小程序：查看皇室战争受欢迎的卡片以及卡组

[数据来源：爬取的statsroyale网站的数据] 数据爬取的可以看我的另一个项目

### 目录结构：
- images — 存放项目图片
- pages — 存放项目页面相关文件
- utils — 存放utils文件，可require引入
